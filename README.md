nido.
Aplicación móvil de búsqueda de inmuebles
1. Resumen
nido es una aplicación móvil (iOS y Android) para búsqueda de pisos en alquiler y compra en
España. Su propuesta de valor se diferencia radicalmente de los portales actuales (Idealista,
Fotocasa, Pisos.com) en tres ejes: experiencia de usuario de nivel superior, filtros
hiperpersonalizados accionables y transparencia radical sobre el inmueble y el entorno.
